# 🎮 BODMAS Learning Game

🔗 **Live Demo:** https://bodmasgame.infinityfreeapp.com/

## 📌 Overview
This project is an interactive game designed to **teach BODMAS (order of operations)** through visual and step-by-step problem solving.

Instead of memorizing rules, users learn by selecting the correct operation at each step with real-time feedback and explanation.

---

## 🚀 Features
- Learn Mode (step-by-step guidance)
- Play Mode (timed challenge)
- Visual highlighting of operations
- Instant feedback (correct/wrong with explanation)
- Score, timer, and levels
- Leaderboard system
